网络通信小程序，利用socket在服务端和客户端之间通信，可相互上传、下载、查询文件和目录

*常用指令*

发送消息
* -s/-send message

查询目录/文件，参数 -mul 为可选，表示递归查看子目录
* -f/-find 目标目录/文件 -mul

上传目录/文件
* -u/-upload 本地目录/文件 目标目录

下载目录/文件
* -d/-download 目标目录/文件 本地目录

切换输入输出方
* -r/-reverse

断开连接
* -q/-quit

显示此帮助
* -h/-help
